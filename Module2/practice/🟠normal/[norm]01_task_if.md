## "Трехзначное ли"

### Задание

Проверьте, является ли данное число трехзначным.

### Формат входных данных

Дано целое число.

### Формат выходных данных

Вывести "Да", если данное число трехзначное и "Нет" в противоположном случае.

### Решение задачи

```python
num = int(input("N: "))
if num >= 100 | num < 1000:
    print("ДА")
else:
    print("Нет")
```
